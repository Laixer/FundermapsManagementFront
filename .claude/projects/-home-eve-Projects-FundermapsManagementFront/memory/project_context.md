This app is a legacy management portal, only used by two people to manage Fundermaps. It was half finished.

**Why:** Understanding the app's status helps calibrate effort and expectations — don't over-engineer, keep changes pragmatic.

**How to apply:** Don't invest heavily in polish, testing infrastructure, or long-term architecture. The app will likely be integrated into the maps app (FunderMaps frontend) as an admin page. The backend it talks to is the Go backend (FunderMapsTest).
